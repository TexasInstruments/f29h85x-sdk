Library Structure

The PMBus Communications Stack Library is distributed as part of the C29 Software Framework. The library and source code are packaged under the following location :

/f29h85x-sdk_1_00_00_00/source/communication/PMBus

 

   

folder name description
ccs contains the projectspec for the PMBus Stack generation
include contains the include files for the source
lib Pre Built Binaries of the files (.lib)
source contains the source files linking to the driverlib to implement transactions

   

While the examples are packed under the folder

/f29h85x-sdk_1_00_00_00/examples/communication/PMBus

 

 

folder name description
pmbus_controller contains the projectspec for the controller example
pmbus_target contains the projectspec for the target example
common/controller comprises of methods/tests that the controller uses for transaction/acknowledgement
common/device_config contains device specific configurations for sysconfig and example setup
common/include example setup header file
common/target comprises of methods/tests that the target uses for transaction/acknowledgement
