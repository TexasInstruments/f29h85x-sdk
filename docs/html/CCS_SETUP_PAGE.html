Note
The steps on this page need to be done once on a given host machine

Download CCS

  • Download CCS 20.3.0 or later from https://www.ti.com/tool/CCSTUDIO
  • Unzip the file for Windows at any location on your host PC

Install CCS

  • Install CCS 20.3.0 by double clicking the installer file from the downloaded and un-zipped CCS package file.

    CCS Installer File

  • Follow the steps and at below screen, recommend to keep install directory as default.

    CCS Install Path

  • Follow the steps and at below screen, select the component as "C2000 real-time microcontrollers" to install F29H85x related emulation and GELs

    CCS Select Components
  • Follow the steps, until CCS is installed.
  • Launch CCS and workspace gets created at "C:\Users\user_id\workspace_ccstheia".

Install C29 compiler "TI CLANG v2.0.0.STS "

  • Download from "TI CLANG v2.0.0.STS " https://www.ti.com/tool/download/C29-CGT/2.0.0.STS
  • Install at "C:\ti\ccs2030\ccs\tools\compiler"
    Install C29 Compiler

Check Packages as seen by CCS

  • Launch CCS

  • Goto "File > Preferences"

    CCS Preferences
  • Goto "Code Composer Studio Settings > General > Products", make sure you see "F29H85X-SDK " listed here.

    • Sometimes, you need to click "Refresh"

      CCS Products (SDK)
  • Goto "Code Composer Studio Settings > General > Products", make sure you see "SysConfig 1.25.0 " listed here.

    • Sometimes, you need to click "Refresh"

      CCS Products (Sysconfig)
  • Goto "Code Composer Studio Settings > General > Compilers", make sure you see "TI CLANG v2.0.0.STS " listed here

    • Sometimes, you need to click "Refresh"

      CCS Compilers
