Configures the number of data nibbles for standard SENT.

Parameters
baseis the base address of the SENT module.
dataNibbleis the number of data nibbles.

This function configures the number of data nibbles for standard SENT.

The dataNibble specifies the number of number of data nibbles should be expected for standard SENT. Some of the enhanced sensors support 7 and 8 nibbles. Valid values can be refered from the enum SENT_DataNibble.

Returns
None.

Configures the number of clock cycle per Tick.

Parameters
baseis the base address of the SENT module.
ttClockis the number of clock cycle per tick.

This function configures the number of clock cycle per tick when generating the internal tick clock. This tick clock is used for the initial calculation of calibration Sync errors (25% differential). The default is 300 (12Ch), which is typically 3 us ticks for 100 Mhz clock.

The ttClock specifies the number of clock cycle per tick. It should be a value between 1 and 65536 clock cycles inclusive.

Returns
None.

Set the CRC checksum calculation.

Parameters
baseis the base address of the SENT module.
rxCRCis the CRC checksum calculation for data nibbles.

This function sets the CRC checksum calculation.

Valid values for rxCRC parameter are:

  • SENT_RX_CRC_WITHOUT_STATUS - CRC checksum calculation for the data nibbles only
  • SENT_RX_CRC_WITH_STATUS - CRC checksum calculation for status nibble and data nibble
Returns
None.
