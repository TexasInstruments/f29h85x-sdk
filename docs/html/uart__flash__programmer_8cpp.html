APIs for Uart Flash Programmer

Introduction

Host-side Flash Programmer cli for firmware upgrades and other uility options via UART.

Note
This is a demonstration cli in tandem with the Texas Instruments UART Flash Kernel. It takes parameter input to send binary applications, images, and other operations to the mcu via UART (COM/tty port).
