Force TMU ROM Parity error

This function forces the TMU ROM Parity Error. The key 0xA5 must be written along with the force bit, otherwise, writes are ignored. The key is cleared immediately after writing, so it must be written again for every change to this bit.

Returns
None

Clear TMU ROM Parity error

This function clears the TMU ROM Parity Error. The key 0xA5 must be written along with a write of 0 to the force bit, otherwise, writes are ignored.

Returns
None
