APIs for CAN Flash Programmer

Introduction

Host-side Flash Programmer cli for firmware upgrades and other uility options via CAN.

Note
This is a demonstration cli in tandem with the Texas Instruments CAN Flash Kernel. It takes parameter input to send binary applications, images, and other operations to the mcu via CAN
