The SDFM driver provides API to configure the SDFM module. Below are the high level features supported by the SDFM driver.

Features Supported

  • Eight external pins (4 data input and 4 clock input pins) per SDFM module
  • Different configurable modulator clock modes supported
  • Four independent, configurable secondary filter (comparator) units per module
    • Four different filter type selection (Sinc1/Sinc2/SincFast/Sinc3) options available
    • Detection of over-value condition, under-value condition, and Threshold-crossing conditions
    • OSR value for comparator filter unit (COSR) programmable from 1 to 32
  • Four independent configurable primary filter (data filter) units per module
    • Four different filter type selection (Sinc1/Sinc2/SincFast/Sinc3) options available
    • OSR value for data filter unit (DOSR) programmable from 1 to 32
    • Ability to enable or disable (or both) individual filter module
    • Ability to synchronize all four independent filters by using the Main Filter Enable (MFE) bit or by using PWM signals
  • Data filter output can be represented in either 16 bits or 32 bits.
  • Programmable mode FIFO for data filter to reduce interrupt overhead
  • PWMs can be used to generate a modulator clock for SDFM
  • Configurable Input Qualification available for both SD-Cx and SD-Dx
  • Ability to use one filter channel clock (SD-C1) to provide clock to other filter clock channels
  • Configurable digital filter available on comparator filter events to blankout comparator events caused by spurious noise

SysConfig Features

Note
It is strongly recommend to use SysConfig where it is available instead of using direct SW API calls. This will help simplify the SW application and also catch common mistakes early in the development cycle.
  • Comparator configurations for Filter type, OSR and detection of over-value/under-value/Threshold-crossing conditions
  • Data filter configurations for Filter type, OSR, PWM configuration for SDSYNC and FIFO settings
  • Frequency and full scale range settings
  • Integrated support for interrupt
