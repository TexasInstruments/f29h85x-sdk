PMBus Library Overview

Integrating the Library into your Project

‍To begin integrating the library into your project follow these steps:

  • Go to the Project Properties and add a path to the device driverlib directory, to the Include Options section of the project properties. This option tells the compiler where to find the device driver header files.This can also be done by integrating the driverlib.lib file into the project. In addition, you must add a path for the PMBus stack interface, for the compiler to find the stack header files.

 

  • Add the communication stack library <device_name>_PMBus_Communication_Stack.lib , Replace the device name in accordance to your chosen device.
