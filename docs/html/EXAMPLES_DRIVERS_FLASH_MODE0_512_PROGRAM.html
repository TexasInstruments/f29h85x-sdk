Introduction

This example demonstrates how to use the various Flash API modes for 512-bit programming in bank mode 0.

The programming modes demonstrated are:

  • Fapi_AutoEccGeneration
  • Fapi_DataAndEcc
  • Fapi_DataOnly
  • Fapi_EccOnly

Watch Variables

  • None

External Connections

  • None.

Steps to Run the Example

  • When using CCS projects to build, import the CCS project for the required combination and build it using the CCS project menu (see CCS Build Project).
  • When using makefiles to build, note the required combination and build using make command (see Makefile based Build)
  • Launch a CCS debug session and set breakpoints after each programming command
  • Using the memory browser, watch flash memory update as you step through the program
