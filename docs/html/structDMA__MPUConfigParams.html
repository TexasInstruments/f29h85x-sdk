DMA_MPUConfigParams Struct Reference

Detailed Description

Values that can be passed to DMA_configMPURegion() as the configure parameter. The startAddr is the beginning address of the 4KB granular block, whose least 12 bits are 0's. The endAddr denotes the end of the granular region and the least 12 bits are 1's (0xFFF)
