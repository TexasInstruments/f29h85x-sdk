void PMBusStack_targetStateHandler ( PMBus_StackHandle  handle)

PMBus Target Stack State Machine Handler

Parameters
handleis the handle to the PMBus stack object

This function implements the state machine of the PMBus in target mode. This handler is designed to operate within the PMBus interrupt service routine (ISR) triggered by the following interrupts:

  • Data Ready (Read buffer is full)
  • Data Request (Controller has requested data) / SDIR bit in I2C
  • EOM (Controller signals an end of a block message)
Note
The handler must be called in the PMBus ISR only
Returns
None.
