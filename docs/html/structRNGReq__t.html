Detailed Description

This is RNG type which holds the resultPtr for derivation which is returned by TIFS. This also holds the resultLengthPtr and DRBG Mode along with seedValue and seedSize.

Parameters
resultPtrPointer to the random number generated
resultLengthPtrPointer to store the desired length in bytes
DRBGModeFlag that determines whether DRBG mode is required or not
seedValueStores the seed values
seedSizeInDWordsStores the seed size in double words
